# 接口：changePassword  [返回](../README.md)
用例： [修改密码](../用例/修改密码.md)

* 权限：
    老师,学生必须登录

* 功能：
   修改密码

* API请求地址：
   接口基本地址/com/api/changePassword

* 请求方式 ：
   POST

* 请求参数说明:
    无

* 返回实例：
    
       {
               "code": 200,                        
               "data": {
                  "PASSWORD":"*****"
                },
               "msg": "操作成功"
            }

* 返回参数说明：

    |参数名称|说明|
    |:---------:|:--------------------------------------------------------|
    |code|请求响应状态|
    |data|所有学生的信息数组|
    |PASSWORD|users密码|
    |msg|操作信息|
